# DualFRA: Dual-stream feature fusion for enhanced Fall Risk Assessment
DualFRA is a novel vision-based deep learning approach that leverages standard cameras and integrates complementary mobility information from both Timed Up and Go (TUG) and Five-Times Sit-to-Stand (FTSTS) test videos for comprehensive fall risk assessment. DUALFRA first applies 3D pose estimation to extract skeletal sequences from the two clinical tests. For each sequence, multiple complementary representations are constructed, including a spatio-temporal skeletal graph and auxiliary views that emphasize joint configurations and motion dynamics. These representations are then processed by a dedicated spatio-temporal feature encoder that combines self-attention-enhanced graph convolution with multi-scale temporal modeling to capture discriminative movement patterns. The resulting dual-stream features are integrated through cross-attention to produce a single, coherent semantic embedding, which is subsequently used for fall risk prediction through a multi-layer perceptron.
